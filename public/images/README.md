## Instruções de Uso
1. Baixe as imagens dos links fornecidos
2. Otimize as imagens para web usando ferramentas como:
   - TinyPNG (https://tinypng.com/)
   - Squoosh (https://squoosh.app/)
3. Redimensione para os tamanhos recomendados
4. Salve na pasta `public/images/`
5. Certifique-se de que os nomes dos arquivos correspondam aos utilizados no código

## Notas
- Todas as imagens são gratuitas para uso comercial
- Mantenha a consistência visual entre as imagens
- Priorize imagens com boa iluminação e composição profissional
- Evite imagens com texto ou marcas d'água
